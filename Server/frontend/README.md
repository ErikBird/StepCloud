# frontend

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Run your tests

```
npm run test
```

### Lints and fixes files

```
npm run lint
```

### Run your unit tests

```
npm run test:unit
```

### To Resolve any error by Segmentation Faults

```bash
 npm cache clean --force
 rm -rf node_modules package-lock.json
 npm install
```
